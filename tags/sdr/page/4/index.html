    Webブラウザで144MHzのアマチュア無線を受信してみる

    RTL2832Uドングルで使うrtl-sdrには、rtl_tcpのほかにもいくつかコマンドが含まれています。そのなかでもrtl_fmというコマンドはその名から予想されるとおり、FMを受信することができます。これはFM放送だけではなく、アマチュア無線のような狭帯域NBFMも復調できます。実はヘルプを見てみるとAMやSSB(LSB/USB)、果てはCWの復調やI/Q信号を取り出すことまで可能なようです。復調周波数や復調モードなどはすべてコマンドライン引数で指定することができます。 Read more …
